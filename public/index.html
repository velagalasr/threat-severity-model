üìö NSL-KDD Dataset

NSL-KDD stands for Network Security Laboratory - Knowledge Discovery in Databases dataset. It's an improved version of the KDD Cup 1999 dataset, widely used for network intrusion detection research.

üö® Attack Categories
DoS (Denial of Service):
Attacks that make computer or network resources unavailable by overwhelming the system with requests. Examples: SYN flood, ping flood, teardrop attacks.
Probe (Surveillance):
Reconnaissance attacks to gather information about network vulnerabilities. Examples: port scanning, network mapping, vulnerability scanning.
R2L (Remote to Local):
Unauthorized access from remote machine to gain local access. Examples: password guessing, FTP write attacks, phishing, social engineering.
U2R (User to Root):
Privilege escalation attacks where attacker gains root/administrator access from normal user account. Examples: buffer overflow, rootkit installation, privilege exploitation.
üéØ Model Purpose
This XGBoost-based model analyzes network traffic patterns and predicts threat severity scores (0-10) to help security teams prioritize incident response. It was trained on 100,778 samples with 61 engineered features including protocol patterns, connection rates, error ratios, and service behaviors.
